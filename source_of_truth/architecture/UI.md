the browser is only here for providing a GUI. all data computation has to be done on the server. 

the UI is one main full screen page. additional windows can be toggled in the top bar navigation. a window can be moved and resized.  

the UI state should be stored in the database. so for example if a window is opened and moved and then the app is reloaded the same windows should show up and with the size and position they had last time 

use vue.js but only use the compositions API!
